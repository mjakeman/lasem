Lasemcairo_set_source_pixbuf

Declaration 

void
lsm_cairo_set_source_pixbuf (
  cairo_t* cairo,
  const GdkPixbuf* pixbuf,
  double pixbuf_x,
  double pixbuf_y
)

Description

Sets the given pixbuf as the source pattern for cairo.

The pattern has an extend mode of CAIRO_EXTEND_NONE and is aligned so that the origin of pixbuf is pixbuf_x, pixbuf_y.

Available since:0.4

Parameters

cairo cairo_t
 

A cairo context.
 The data is owned by the caller of the function.
pixbuf GdkPixbuf
 

A GdkPixbuf.
 The data is owned by the caller of the function.
pixbuf_x double
 

X coordinate of location to place upper left corner of pixbuf.
pixbuf_y double
 

Y coordinate of location to place upper left corner of pixbuf.
